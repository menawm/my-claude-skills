## LLM Council

Runs your question through 5 independent AI advisors who each approach it from a different angle, peer-review each other, and deliver a final synthesized verdict.

Built by [Ole Lehmann](https://x.com/itsolelehmann). Based on Andrej Karpathy's [LLM Council](https://github.com/karpathy/llm-council) methodology.
